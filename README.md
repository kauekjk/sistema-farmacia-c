# Sistema de Gestão de Estoque para Farmácia
Projeto acadêmico desenvolvido em C para praticar estruturas de dados e ponteiros, simulando um módulo de gerenciamento de estoque de medicamentos

## Funcionalidades

- Cadastro de medicamentos
- Registro de quantidade inicial em estoque
- Atualização da quantidade utilizando ponteiros
- Exibição do relatório final do estoque

## tecnologias
- Linguagem C

## Conceitos aplicados
- Structs
- Ponteiros
- Manipulação de memória
- Entrada e saída de dados
- Organização de código

## Como executar
1. Compile o arquivo:
```bash
gcc main.c -o farmacia
```

2. Execute:
```bash
./farmacia
```

## Objetivo

Aplicar os conceitos de estruturas de dados e ponteiros em C por meio da implementação de um sistema simples de gerenciamento de estoque.
